# America's Broadband Problem

Author: Mintay Misgano, PhD

This project examines county-level variation in U.S. broadband access using public data from IMLS, BroadbandNow, FCC-linked indicators, ACS-based measures, and county geodata. It was completed as graduate coursework in data visualization and public-policy analysis, with Python and RapidMiner used for data preparation and Tableau used for the final interactive story.

## Key Findings

- County-level analysis reveals broadband gaps that state and national averages can hide.
- Poverty rate is more informative than FCC-style availability measures when interpreting likely real-world access.
- Broadband outcomes follow clear regional and rural-urban patterns rather than being evenly distributed.
- Counties with higher concentrations of Black residents are overrepresented among poorer broadband outcomes in this dataset.
