# Advanced Regression Methods

This project brings together several regression models that become necessary when the dependent variable does not meet the assumptions of ordinary linear regression. Instead of treating regression as one default tool, the collection shows how the structure of the outcome variable determines the model that should be used.

The methods span several common cases: count data, overdispersed counts, ordered categories, truncated samples, and censored outcomes. Seeing them together makes the logic of model choice much clearer than encountering them as isolated techniques.

One of the strengths of the collection is that it pairs related models where comparison matters. Poisson and negative binomial regression show how overdispersion changes the appropriate count model, while truncated and Tobit regression illustrate two different ways that restricted outcomes alter estimation.

As a portfolio piece, this collection demonstrates advanced regression breadth, strong statistical judgment about when OLS is not appropriate, and the ability to package specialized methods into a coherent public-facing reference.
