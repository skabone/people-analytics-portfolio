# R Statistical Methods Showcase

This project brings together several core statistical methods from graduate coursework in R and applies them to a shared time-to-event dataset. Instead of treating each method as a separate lab exercise, the collection shows how descriptive analysis, hypothesis testing, regression, and survival modeling can be positioned within one coherent analytical workflow.

The strongest feature of the collection is the inclusion of survival analysis alongside more familiar methods such as t-tests, ANOVA, and regression. Kaplan-Meier and Cox proportional hazards models make the project more than a general statistics review; they show how time-to-event outcomes can be handled directly rather than reduced to simpler snapshots.

Using one dataset across the full sequence makes the differences among methods easier to understand. Some methods test group differences, others model continuous or binary outcomes, and survival models focus on timing and censoring. That contrast gives the project real instructional value as a compact R methods reference.

As a portfolio piece, this collection demonstrates statistical breadth, R-based reproducible analysis, and the ability to organize course material into a more coherent public-facing artifact.
