# Python ML Methods Showcase

This project brings together eight machine learning methods from graduate coursework into one comparative workflow. Instead of presenting each lab separately, the project applies them to the same structured dataset so differences in model behavior, evaluation, and interpretability are easier to see side by side.

The supervised portion includes linear regression, logistic regression, decision trees, support vector machines, and random forest models. The unsupervised portion adds k-means clustering, Gaussian mixture modeling, and principal components analysis. Using one dataset across all of them makes the collection useful as a compact reference point for how core Python ML methods behave on a shared problem.

The strongest value of the project is comparative clarity. Random forest produced the best classification performance in this workflow, while logistic regression and decision trees remained easier to interpret. The clustering and PCA sections add a second layer by showing how the same dataset can be explored structurally rather than only predicted from labels.

As a portfolio piece, this collection demonstrates breadth across core machine learning methods, consistent evaluation habits, and the ability to organize coursework into a coherent, reusable technical artifact.
