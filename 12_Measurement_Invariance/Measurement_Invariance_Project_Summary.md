# Measurement Invariance Testing of the AMS

This project evaluates whether the Ableist Microaggressions Scale can be compared fairly across disability-severity groups. The core question is not just whether the scale has a stable factor structure, but whether respondents in different groups interpret and endorse items in sufficiently similar ways for group comparisons to be meaningful.

The analysis follows the standard measurement-invariance sequence through configural, weak, and strong invariance testing. That structure makes the project a useful example of how psychometric evaluation extends beyond ordinary CFA when a scale is used for cross-group comparisons.

The key result is that weak invariance holds but strong invariance does not. In practical terms, the constructs appear to be measured on the same metric across groups, but the item intercepts differ enough that raw observed means cannot be treated as directly comparable without additional modeling.

As a portfolio piece, this project demonstrates multi-group CFA, fit-change interpretation, and the ability to connect psychometric results to concrete decisions about how survey scores should and should not be used.
