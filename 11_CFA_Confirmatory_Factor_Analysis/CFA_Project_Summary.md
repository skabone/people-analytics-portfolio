# Confirmatory Factor Analysis of the GRMSAAW

This project evaluates whether the Gendered Racial Microaggressions Scale for Asian American Women is best represented as one general factor or as four correlated subscales. Using confirmatory factor analysis, the project compares these competing structures directly and shows how fit statistics can be used to justify one measurement model over another.

The strongest value of the project is its combination of psychometric reasoning and technical replication. The analysis is implemented in both R and Python, which makes it useful not only as a CFA example, but also as a cross-platform validation workflow. The same measurement question is answered twice with closely aligned results.

The outcome is clear: the four-factor model fits substantially better than the one-factor alternative. That matters because it changes how the instrument should be scored and interpreted. A scale that behaves as a multidimensional measure should not be collapsed into a single score without evidence that doing so is justified.

As a portfolio piece, this project demonstrates psychometric model testing, fit-index interpretation, and reproducible CFA workflows across software environments.
